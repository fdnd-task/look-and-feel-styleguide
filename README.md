# Styleguide

Ontwerp en maak met een team een styleguide om te gebruiken voor een opdracht van een opdrachtgever.

## Afspraken 
Classnames 
- Taal Engels.
- BEM principes gebruiken. (block, element, modifier) header-link-button:hover
- beschrijf wat het element doet, en niet hoe het eruit ziet

Code conventies  
- Maak netter code. 
- geen AI. 
- witregels tussen componenten.
- lowercase namen voor class etc.
- alleen <div> gebruiken om onderdelen te deviden.
- voeg alt

Hoe tegelijk samenwerken in 1 repo
- extensie: liveshare
- Hoofdsharer commit alles, die moet co-authors toevoegen.
